TODO App V1: This project is built using HTML, CSS, JavaScript with ReactJS Library.

- Main Features:
1. Add a task:
  a.  Due date is chosen from the modern Calendar
2. Complete a task:
  a.  Animated checkbox icon
  b. Cross line when marked as completed
3. Remove a task:
  a. Animated trash icon
4. Edit task (additional feature):
  a. Animated edit icon
  b. Modify the task in-place 

- Run the Project:
  o You can run the project in development mode with the following commands: 
     1. npm install
     2. npm start
  o Open http://localhost:3000 to view it in the browser.
